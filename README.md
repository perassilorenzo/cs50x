# CS50x – My Learning Journey

Welcome to my CS50x repository.

This repository contains my complete learning journey through CS50x, Harvard University's introduction to Computer Science. Here I collect my notes, exercises, projects, resources, and progress throughout the course.

The goal of this repository is not only to document what I learn, but also to build a structured knowledge base that I can revisit in the future.

## About CS50x

CS50x is Harvard University's introduction to the intellectual enterprises of computer science and the art of programming.

The course starts from fundamental concepts such as computational thinking, algorithms, and problem solving, and gradually introduces programming languages and technologies including:

- Scratch
- C
- Python
- SQL
- HTML
- CSS
- JavaScript

The focus is not only on writing code, but on learning how to think computationally and solve problems efficiently.

## Why I'm Taking CS50x

As an IT student, I often find that topics are taught separately and without a clear connection between them.

By following CS50x, I want to build a stronger understanding of computer science fundamentals, connect concepts together, improve my problem-solving skills, and develop a more structured way of thinking about software and technology.
